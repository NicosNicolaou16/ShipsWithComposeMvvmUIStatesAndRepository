# Ships With Compose Mvvm UI States and Repository
The project is a test project that implements and test JetPack Compose with UI States, Hilt and Room Database.

# This Project contains:
Compose <br />
Hilt <br />
Room Database <br />
Coroutines <br />
Kotlin ktx <br />
MVVM <br />
Support kapt and ksp (ksp only setup for Room Database) <br />
minifyEnabled, shrinkResources, R8 are enabled <br />
Gradle Kotlin DSL <br />
Repository <br />
UI States <br />
Offline <br />
